STEP 1: COMPILE THE PROJECT
---------------------------
From the project root directory (term-ii-project-submission-tanishac2020):

Option A - Using PowerShell:
  javac -d bin src/com/library/model/*.java src/com/library/service/*.java src/com/library/main/*.java

Option B - From src directory:
  cd src
  javac -d ..\bin com/library/model/*.java com/library/service/*.java com/library/main/*.java
  cd ..

STEP 2: RUN THE APPLICATION
----------------------------
From the project root directory:

  java -cp bin com.library.main.Main

TROUBLESHOOTING:
----------------
1. "javac is not recognized":
   - Make sure Java JDK is installed
   - Add Java bin folder to PATH environment variable
   - Restart command prompt/PowerShell

2. "Class not found" error when running:
   - Check that bin directory has been created with .class files
   - Verify classpath is correct: -cp bin

3. "Package not found" error during compilation:
   - Check that folder structure matches the package structure
   - Run javac from the project root directory

4. Error during menu input:
   - Enter numeric values only for ID and menu choices
   - Enter text strings when prompted for book titles, authors, or names
